# ⚡ Max's Anime Soul Site

**THE POWER OF THE THREE GREATS AWAKENS THE LEGENDS**

An interactive, motion-graphics-rich anime website featuring the legendary trio: **Gojo Satoru**, **Tanjiro Kamado**, and **Max — The Creative Legend**.

## Features

- 🎨 Hand-drawn SVG character silhouettes (Gojo, Max, Tanjiro)
- ✨ Animated stat cards with shimmer effects
- 📖 Manga preview cards with page-turn transitions
- 🔥 Rising ember particle system
- 💫 Scroll-triggered reveal animations
- 🤫 **Secret Ultra Super Mode** — type the code to unlock legendary power
- 📱 Fully responsive — desktop, tablet, and mobile

## Tech Stack

- React + Vite
- CSS animations & keyframes (no animation libraries)
- SVG character art
- IntersectionObserver scroll reveals

## Getting Started

```bash
npm install
npm run dev
```

## Deploy

Connected to Vercel for automatic deployments.
